메소드 (함수 만들기)
private static void showUpNewSentence()
{
	System.Console.WriteLine("새로운 문장");
}

private		-> 접근 통제, 자신만
protected	-> 자신과 자식만
public		-> 모두 개방

메소드로 값을 반환하고 싶으면 void 자리에 반환할 값의
자료형을 적고 메소드 마지막에 return 반환할 값; 적기
